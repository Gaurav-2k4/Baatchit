📱 Baatchit

Baatchit is a modern Flutter-based mobile application designed to provide a smooth, fast, and user-friendly chatting experience. The app is built with a clean architecture and focuses on performance, scalability, and an intuitive UI.

🚀 Features
Cross-platform support (Android & iOS)
Fast and responsive UI
Clean and modular code structure
Easy to extend and customize
🛠️ Tech Stack
Flutter
Dart
