📱 Baatchit <br>

Baatchit is a modern Flutter-based mobile application designed to provide a smooth, fast, and user-friendly chatting experience. The app is built with a clean architecture and focuses on performance, scalability, and an intuitive UI. <br>

🚀 Features <br>
Cross-platform support (Android & iOS)  <br>
Fast and responsive UI <br>
Clean and modular code structure <br>
Easy to extend and customize <br>

🛠️ Tech Stack <br>
Flutter <br>
Dart <br>
